# XAPI

XAPI is optimized Api with Fastify Typescript with manual typings

## Installation

- Compile the entire project use Typescript

```bash
tsc
```

- Copy _nsfw_model_ to _dist_

## Usage

```bash
sh start.sh
```

## Documentation

- open swagger.yaml with your openApi compiler
